============================================================
VERCEL STATIC IP NOTE
============================================================

Vercel Static IPs are for outbound egress traffic from deployments.
After Static IPs are enabled for the Vercel project, Vercel provides
static outbound IP pairs that can be shared with external services for
allowlisting.

Until Static IPs or another fixed egress route is enabled, the production
POS server does not have a guaranteed single public outbound IP address
that is safe to give to INET for long-term allowlisting.
